# awesome-composed-image-retrieval

A curated list of papers on **Composed Image Retrieval (CIR)**,  
where an image is modified by text to retrieve a target image.



## Papers


| Year | Title | Links |
|------|-------|-------|
| CVPR'25 | ConText-CIR: Learning from Concepts in Text for Composed Image Retrieval | [code](https://github.com/mvrl/ConText-CIR)  [paper](https://arxiv.org/pdf/2505.20764) |
| CVPR'25 | CoLLM: Integrating Collaborative Embeddings into Large Language Models for Recommendation | [code](https://collm-cvpr25.github.io)  [paper](https://openaccess.thecvf.com/content/CVPR2025/papers/Huynh_CoLLM_A_Large_Language_Model_for_Composed_Image_Retrieval_CVPR_2025_paper.pdf) |
| NIPS'25 | Towards Robust Uncertainty Calibration for Composed Image Retrieval | [paper](https://neurips.cc/virtual/2025/loc/san-diego/poster/118555) |
| NIPS'25 | i-CIR: Instance-Level Composed Image Retrieval | [code](https://github.com/billpsomas/icir?tab=readme-ov-file) [paper](https://arxiv.org/pdf/2510.253875) |
